# User Stories

1. As a buyer, I want to create a purchase requisition so I can request materials through SAP.
2. As a procurement officer, I want real-time status updates when SAP creates or modifies a PO so I can track fulfillment.
3. As a finance analyst, I want a dashboard showing vendor spend and approval cycle times to improve procurement strategy.
4. As a system admin, I want all data synchronized via secure OAuth-based integration to ensure compliance.
5. As an executive, I want a demo-ready SaaS architecture that showcases cloud-native SAP integration capabilities.
